## Предметы
**Осколок Тьмы** падает с мобов, основной ресурс для крафта.
**Флакон Теней** выпей ПКМ, получишь невидимость, скорость и ночное зрение. И тошноту.
**Печать Алхимика** ПКМ: +10 сердец и реген. Одноразовая, исчезает после использования.
**Камень Бездны** падает с боссов. Переплавляется в незерит.
**Амулет Теней**  пока не реализова айтем просто текстураа.

## Крафты

Флакон: 4 осколка вокруг бутылки с водой  
Печать: золото + осколки + алмаз (3x3)  
Камень: 8 осколков + незеритовый лом (3x3)  
Амулет: осколки + жемчуг края + камень бездны (3x3)

## Установка

Fabric 1.20.1 + KubeJS  скопировать папку `kubejs/` в инстанс.

команда `/shadowinfo` — список предметов в игре.
